# Seismic Maintenance and Inspection: Low-Cost Risk Reduction

Regular maintenance and inspection are often overlooked but can prevent catastrophic failures at near-zero cost. This document covers the essential maintenance items that improve seismic resilience.

## The Maintenance-Seismic Connection

Deferred maintenance degrades seismic capacity:
- **Rusted connections** → brittle fracture instead of ductile yielding
- **Cracked infill** → lost stiffness → altered load path
- **Corroded rebar** → reduced section → premature failure
- **Water damage** → mortar deterioration → loss of bond

## Routine Inspection Checklist (Annual)

| Item | What to Check | Action if Found |
|------|---------------|-----------------|
| **Wall cracks** | Diagonal, stair-step, horizontal | Monitor width; >3mm → engineer review |
| **Foundation** | Settlement, cracking, moisture | Drainage repair; underpin if progressive |
| **Roof-to-wall** | Visible anchors, straps, bolts | Tighten loose; replace corroded |
| **Parapets/chimneys** | Cracks, lean, loose masonry | Brace or remove if unstable |
| **Façade elements** | Cladding, cornices, signs | Secure or remove if hazardous |
| **Stair/elevator shafts** | Cracks at connections | Verify structural separation |
| **Utility attachments** | Tanks, HVAC, piping | Flexible connections; seismic restraints |
| **Fire protection** | Sprinkler pipe sway braces | Add if missing |

## Post-Earthquake Inspection (ATC-20)

| Placard | Meaning | Action |
|---------|---------|--------|
| **Green (Inspected)** | No apparent hazard | Normal occupancy |
| **Yellow (Restricted Use)** | Hazardous condition | Limited entry; repairs needed |
| **Red (Unsafe)** | Imminent collapse danger | **Do not enter**; evacuate |

**Every building should have ATC-20 forms and trained personnel before an earthquake.**

## Maintenance Priorities by Building Type

| Type | Critical Maintenance |
|------|---------------------|
| **URM** | Mortar repointing (lime-based); wall anchors; parapet bracing |
| **RC Non-ductile** | Cover concrete repair; corrosion inhibitor; joint sealant |
| **Steel** | Connection bolt check; corrosion protection; bracing inspection |
| **Wood** | Foundation bolts; hold-downs; sheathing nails; moisture control |
| **All** | **Non-structural**: ceilings, partitions, glazing, equipment |

## Low-Cost, High-Impact Maintenance Items

| Item | Cost | Seismic Benefit |
|------|------|-----------------|
| **Repoint mortar joints** (lime mortar) | Low | Restores wall integrity; prevents water ingress |
| **Tighten anchor bolts/nuts** | Very low | Restores wall-diaphragm connection |
| **Brace water heater / tall furniture** | Very low | Prevents injury/fire |
| **Install cabinet latches** | Very low | Reduces falling hazards |
| **Secure suspended ceilings/lights** | Low | Prevents overhead falling hazards |
| **Flexible gas/electric connections** | Low | Prevents post-EQ fires |
| **Clean roof drains / gutters** | Very low | Prevents water damage → mortar decay |

## Myanmar Context

| Challenge | Maintenance Approach |
|-----------|---------------------|
| Seasonal flooding (monsoon) | **Pre-monsoon**: check drainage, foundation, waterproofing |
| Termite damage (timber) | Annual inspection; borate treatment; replace damaged members |
| Corrosion (coastal/humid) | Annual coating check; cathodic protection for critical steel |
| Informal additions | Identify unauthorized floors/rooms; assess load path |

## Cost-Benefit

| Annual Maintenance Budget | Risk Reduction |
|---------------------------|----------------|
| 0.5–1% of replacement value | Prevents 20–40% of avoidable seismic damage |
| Post-EQ rapid assessment | Reduces casualties by enabling safe re-entry |
